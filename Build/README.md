# Dossier build/

Ce dossier contient tout ce qui est lié à la construction physique du projet LanzarMAXP124D.

## Contenu
- **cut-list.txt** : liste des panneaux à découper.
- **assembly-guide.md** : étapes d’assemblage du caisson.
- **bracing-plan.md** : plan des renforts internes.

## Objectif
Fournir toutes les informations nécessaires pour fabriquer le caisson : dimensions, montage, renforts et recommandations pratiques.
