# Plan de renforts – LanzarMAXP124D

## Objectif

Limiter les vibrations des panneaux et rigidifier la caisse pour supporter la pression générée par 2× 12" SPL.

## Types de renforts

- **Renfort central** :
  - Entre les flancs gauche/droite.
  - Option : croix reliant haut/bas/flancs.

- **Renforts latéraux** :
  - Entre fond et baffle avant.
  - Possibles renforts verticaux sous chaque sub.

- **Double baffle** (optionnel mais recommandé) :
  - Deux couches de CP en façade pour les subs.

## Recommandations

- Percer des ouvertures dans les renforts pour laisser circuler l’air.
- Éviter de bloquer le flux d’air vers les évents.
- Coller + visser tous les renforts.
