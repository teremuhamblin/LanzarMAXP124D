# Guide d’assemblage – LanzarMAXP124D

## Étapes principales

1. **Préparation des panneaux**
   - Découper tous les panneaux selon la cut-list.
   - Vérifier l’équerrage.

2. **Montage de la caisse**
   - Assembler fond + flancs + haut + bas.
   - Coller + visser (pré-perçage recommandé).

3. **Renforts**
   - Installer les renforts internes (croix, latéraux).
   - Vérifier qu’aucun renfort ne gêne les subs ou les évents.

4. **Baffle avant**
   - Découper les ouvertures pour les 2× 12".
   - Prévoir les trous pour les évents si en façade.
   - Fixer le baffle (collage + vis).

5. **Étanchéité**
   - Appliquer du mastic ou silicone dans toutes les jonctions internes.
   - Vérifier l’absence de fuites.

6. **Montage des subwoofers**
   - Câbler les bobines selon `docs/wiring-4ohm-final.md`.
   - Fixer les subs avec vis à bois ou inserts.

7. **Tests**
   - Mesurer l’impédance.
   - Faire un test à faible volume.
   - Monter progressivement en puissance.

## Conseils

- Toujours vérifier la polarité.
- Ne jamais pousser à fond sans rodage minimal.
- Surveiller la chauffe de l’ampli lors des premiers tests.
