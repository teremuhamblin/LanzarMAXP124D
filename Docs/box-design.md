# Design du caisson – LanzarMAXP124D

## Objectif

Concevoir un caisson bass-reflex pour **2× Lanzar MAXP124D**, orienté SPL / usage muscial agressif.

## Type de caisson

- Type : Bass-reflex
- Orientation : SPL / street / daily
- Nombre de chambres :
  - Option 1 : chambre commune
  - Option 2 : chambres séparées (plus safe en cas de casse d’un sub)

## Volume et accord (base de travail)

- Volume brut total : 110–130 L
- Volume net visé (après subs, renforts, évents) : ~90–100 L
- Accord : 32–38 Hz selon usage
  - 32–34 Hz : plus profond, plus musical
  - 36–38 Hz : plus SPL, plus d’impact

## Matériaux

- Panneaux : CP bouleau 18 ou 21 mm
- Baffle avant : possible double épaisseur
- Renforts : internes en croix + latéraux

## Évents

- Type : slot port (recommandé) ou évents circulaires
- Grande section pour limiter les bruits d’écoulement
- Longueur à ajuster selon la fréquence d’accord choisie

## Points à affiner

- Volume exact par simulation (WinISD)
- Longueur et section des évents
- Position des renforts pour limiter les vibrations
