# Matching ampli – LanzarMAXP124D

## Impédance et puissance

- Impédance finale du caisson : **4Ω**
- Puissance totale visée : **1200–1600W RMS**

## Type d’ampli recommandé

- Ampli mono (monoblock) ou bridge stable à 4Ω
- Puissance recommandée :
  - 1200–1500W RMS @ 4Ω pour usage soutenu
  - Jusqu’à 2000W RMS @ 4Ω pour usage SPL ponctuel (avec réglages propres)

## Points importants

- L’ampli doit être **clairement stable à 4Ω** en continu.
- Prévoir :
  - Alimentation correcte (section câbles, batterie, alternateur si car audio)
  - Masse propre
  - Fusibles adaptés

## Réglages recommandés

- Filtre passe-bas : 60–80 Hz (selon système)
- Filtre subsonic : 20–28 Hz (selon accord du caisson)
- Gain : réglé à l’oscilloscope ou au moins sans clipping audible
- EQ : à ajuster avec les presets DSP (voir `src/dsp/spl-tuning-presets.json`)
