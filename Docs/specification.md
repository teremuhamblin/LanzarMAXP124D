# Spécifications – Lanzar MAXP124D

## Description

Le Lanzar MAXP124D est un subwoofer 12" double bobine (DVC) 4Ω, conçu pour les applications SPL et les systèmes haute puissance.

## Caractéristiques principales

- Diamètre : 12"
- Type : DVC (Dual Voice Coil)
- Impédance par bobine : 4Ω
- Impédance possible par sub :
  - 2Ω (bobines en parallèle)
  - 8Ω (bobines en série)
- Puissance :
  - RMS estimée : 600–800W
  - Max : 1600W (marketing)

## Avantages

- Flexibilité de câblage (1Ω, 2Ω, 4Ω, 8Ω selon montage global)
- Construction robuste
- Adapté aux caissons SPL bass-reflex

## Utilisation dans ce projet

- Nombre de subs : 2
- Câblage retenu :
  - Par sub : bobines en série → 8Ω
  - Deux subs 8Ω en parallèle → 4Ω final
- Puissance totale visée : 1200–1600W RMS
