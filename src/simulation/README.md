# Dossier simulation/

Ce dossier contient les fichiers liés aux simulations acoustiques du projet LanzarMAXP124D.

## Objectif
Fournir les données nécessaires pour analyser et optimiser :
- le volume du caisson,
- la fréquence d’accord,
- la réponse en fréquence,
- la vitesse d’air dans les évents,
- le comportement des subwoofers en charge bass-reflex.

## Contenu
- **winisd/** : fichiers de projet WinISD (.wpr) et exports graphiques.
- Courbes possibles :
  - SPL / réponse en fréquence
  - Excursion du cône
  - Vitesse d’air dans l’évent
  - Phase
  - Group delay

## Utilisation
Importer les fichiers WinISD dans le logiciel pour :
- ajuster le volume interne,
- tester différentes fréquences d’accord,
- vérifier la tenue en puissance,
- optimiser le caisson pour SPL ou daily.

## Notes
Les simulations servent de base théorique.  
Les résultats finaux peuvent varier selon :
- l’ampli utilisé,
- le véhicule / pièce,
- l’acoustique réelle,
- les réglages DSP.
