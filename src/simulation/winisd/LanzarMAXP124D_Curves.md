# Courbes WinISD – LanzarMAXP124D

## Courbes à analyser

- SPL / réponse en fréquence
- Excursion du cône
- Vitesse d’air dans l’évent
- Group delay
- Phase

## Daily vs SPL

- **Daily (LanzarMAXP124D_DailyBox.wbx)** :
  - Vb ≈ 95 L
  - Fb ≈ 34 Hz
  - Plus de profondeur, usage musique / daily.

- **SPL (LanzarMAXP124D_SPLBox.wbx)** :
  - Vb ≈ 90 L
  - Fb ≈ 38 Hz
  - Plus d’impact, centré sur la zone SPL.

## Recommandation

- Tester les deux configs dans WinISD.
- Vérifier :
  - Excursion à pleine puissance.
  - Vitesse d’air dans l’évent (< 30 m/s idéalement).
  - Group delay acceptable dans la zone 30–60 Hz.
